heimdall_TakePhotosAlongPath

Type: object

Instruct drone to take photos at the coordinates specified by the path
Example:

"heimdall_TakePhotosAlongPath: {mode: 'SINGLE', interval: 10, gimbal_pitch: -45.0}"

root mode
Type: enum (of string) Default: "SINGLE"

Photo Mode

Must be one of:

  • "BURST"
  • "TIME"

Type: number Default: 5

Interval between photos (in seconds or meters)

Value must be greater or equal to 1

root gimbal_pitch
Type: number Default: -90.0

The angle of the gimbal

Type: number Default: 0.0
